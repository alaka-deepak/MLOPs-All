## About This Repository   - Data Science Projects, MLOps

This repository contains the hands-on projects and experiments I completed while learning MLOps in Data Science projects.

The implementations follow the practical tutorials from Udemy courses. Through these exercises, I practiced key MLOps concepts including:

- Data preprocessing
- Model training and evaluation
- Experiment tracking
- ETL/ELT
- Pipeline creation
- Model deployment
- MLOps best practices

The goal of this repository is to document my learning journey and practical implementation of MLOps workflows in real-world Data Science projects.
